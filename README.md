# Pale Vigil

A horror monster-binding RPG built in React. Explore Hollow Vale, bind Dreads,
defeat the six King's Guard, and face the Hollow King to relight the Vigil Flame.

## Run locally
```bash
npm install
npm run dev
```
Then open the localhost URL it prints.

## Build
```bash
npm run build
```
Output goes to `dist/`.

## Controls
- Move: WASD or arrow keys (hold to walk)
- Advance text / confirm: Space, Enter, or click
- Walk into people to talk
- Progress auto-saves to your browser
